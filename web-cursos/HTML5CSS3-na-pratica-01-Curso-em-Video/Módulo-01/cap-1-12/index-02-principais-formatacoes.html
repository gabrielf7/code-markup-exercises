Principais Formatações

HTML \ Semântica \ Tag precisa fazer sentido para ser semântica.

Negrito / Destaque

Nesta frase existe o termo em negrito usando a tag "b" (não semântico).

Nesta frase existe o termo em negrito usando a tag "strong" (semântico).

Itálico / Ênfase

Nesta frase existe o termo em itálico usando a tag "i" (não semântico).

Nesta frase existe o termo em itálico usando a tag "em" (semântico).

Texto Marcado

Podemos criar também um texto marcado usando a tag "mark" (semântico).

Podemos criar também segundo texto marcado usando a tag "mark" (semântico).

Podemos criar também terceiro texto marcado usando a tag "mark" (semântico).

Texto Grande e Pequeno

Temos aqui um texto grande que é exibido pela tag "big" (Obsoleto).

Temos aqui um texto pequeno que é exibido pela tag "small" (semântico).

Texto Deletado e Inserido

Podemos marcar um texto excluído para indicar que ele deve ser lido, mas não considerado. Para isso, usa-se a tag "del" (semântico).

Podemos marcar um texto inserido para dar uma ênfase (sublinhado) e indicar que ele foi adicionado depois. Para isso, usa-se a tag "ins" (semântico).

Podemos marcar um texto inserido para dar uma ênfase (sublinhado) e indicar que ele foi adicionado depois. Para isso, usa-se a tag "u" (não semântico).

Texto Sobrescrito e Subscrito

Para inserir coisas do tipo x20+3 como texto sobrescrito, é usado a tag "sup" (semântico).

Para inserir coisas do tipo H20 como texto subscrito, é usado a tag "sub" (semântico).
